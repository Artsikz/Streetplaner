\doxysubsection{Detailed Description}
Repräsentiert eine Landkarte mit Städten und Straßen. 

Diese Klasse implementiert eine Karte, die Städte (City-\/\+Objekte) und Straßen (Street-\/\+Objekte) verwaltet. Sie ermöglicht das Hinzufügen, Finden und Zeichnen von Städten und Straßen innerhalb einer grafischen Szene (QGraphics\+Scene).

Die Map-\/\+Klasse erbt von \doxylink{class_abstract_map}{Abstract\+Map} und implementiert dort definierte virtuelle Methoden.
